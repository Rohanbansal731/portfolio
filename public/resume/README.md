# Resume Folder

Place your PDF resume file in this folder.

## Instructions:
1. Add your resume PDF file to this folder (e.g., `Rohan_Bansal_Resume.pdf`)
2. Update the Hero component to reference the correct filename
3. The download link will work automatically

## Example files:
- `Rohan_Bansal_Resume.pdf`
- `resume.pdf`
- `CV_Rohan_Bansal.pdf`

Make sure the filename in the Hero component matches your actual PDF filename.
